# ManaVillage 現行システム機能まとめ

最終更新: 2026-06-22（v1.1「90日間AIメンターシップ・サブスクリプション化」反映後）

## 1. システム概要

英語学習プラットフォーム。クリエイター（メンター）がAIメンターキャラクターと90日間の伴走コースを作成し、学習者が購入・サブスクリプションして学習を進める仕組み。AIによるデイリーチャット伴走、Tier別の講師添削、Stripe決済を中核とする。

### 技術スタック
- **Frontend**: Next.js（App Router, TypeScript, Tailwind CSS）— `Frontend/`
- **Backend**: FastAPI + SQLAlchemy + MySQL — `Backend/`
- **Discord Bot**: discord.py — `DiscordSystem/`（旧機能。現行のメイン認証はDiscord OAuthではない）
- **決済**: Stripe（単発購入・月額サブスクリプション・Webhook）

### 認証方式
- ユーザー名＋パスワード（FastAPI OAuth2 PasswordBearer）
- 管理者は2段階認証（メール認証コード、10分有効）
- Discord OAuth・他ソーシャルログインは未実装

## 2. ユーザー種別（role）

- **学習者（learner）**: コースを購入・サブスクライブし学習を進める
- **クリエイター（creator）**: AIメンターキャラクターと90日コースを作成・運用し収益を得る
- **管理者（admin）**: クリエイター審査・コース管理・通報対応

## 3. 主要機能

### コース・購入
- クリエイター検索・マーケットプレイス（`/creators`）
- コース詳細・購入画面（単発購入 or Tier A/B月額サブスクリプション）
- Stripe Checkout・サブスクリプション開始/解約/Tier変更/Webhook処理
- 料金プラン表（試験別問題集の単価設定、スタータープランによるクレジット付与）

### 90日伴走コース
- クリエイターによる90日コース新規作成（Tier A/B価格設定）
- 日単位コンテンツのAI自動生成（テーマ・タスク・朝昼夜メッセージ）をバックグラウンド処理
- クリエイターの人格プロファイル（AIインタビューで抽出）をAI生成プロンプトに反映
- コース90日カレンダーでのクリエイター編集UI
- Day1初回診断テスト→パーソナライズロードマップ生成
- 学習者の日単位進捗ログ・完了マーク

### デイリー伴走チャット
- 学習者がコースに対して質問→AIが回答（関連コンテンツへのリンク提示）
- **Tier A**: AIのみが回答
- **Tier B**: AIが下書き作成→クリエイターが確認・編集してから学習者へ送信
- 質問のカテゴリー自動分類、フラストレーション検知（学習者の不満兆候を検知し、Tier Bへのアップグレード導線を表示）
- クリエイター側：未回答質問一覧（Inbox）、24時間以上未対応の警告表示、質問分析ダッシュボード

### AIコンテンツ生成スタジオ
- キャラクター（AIメンター）の作成・編集・画像・プレビュー
- スクリプト・ボイス付きコンテンツ・下書きの生成
- AIインタビューによるクリエイターの人格プロファイル抽出（コミュニケーションスタイル・コーチングスタイルなど）

### クリエイター機能
- クリエイター登録申請・プロフィール管理
- 収益ダッシュボード（コース売上・Tier B講師報酬）
- お気に入りクリエイター（学習者側）

### 通知
- 朝夜の励ましメッセージ（JST対応）
- 通知一覧・既読管理

### 管理者機能
- クリエイター申請の承認/審査
- コース停止（一時停止フラグ）
- 通報管理
- Tier B未対応コース監視

### レビュー
- コースレビュー閲覧・投稿

| chat | `/chat` | デイリー伴走チャット・講師承認フロー |
